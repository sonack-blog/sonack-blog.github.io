LeNet-5

Yann LeCun

  • 输入: 32x32
  • Ci: 卷积层
  • Si: 下采样层(池化层)
  • Fi: 全连接层

Note: Ci 默认stride=1

Layer Description Output Notes
INPUT 32x32 32x32 -
C1 卷积层 6x5x5(filter_num x filter_size x filter_size) 6x28x28(CxHxW) 参数量=5x5x6+1(1为bias)
S2 下采样层 2x2,s=2 6x14x14 None
C3 16x5x5 16x10x10 -
S4 2x2,s=2 16x5x5 -
C5 120x5x5 120x1x1 全连接
F6 84 84 7x12的比特图编码
OUTPUT 10 10 -

输出层:

  1. Output层也是全连接层，共有10个节点，分别代表数字0到9。如果第i个节点的值为0，则表示网络识别的结果是数字i。
  2. 采用径向基函数(RBF)网络连接方式

    上式中的Wij的值由i的比特图编码确定，i从0到9，j取值从0到7×12-1。
    RBF输出的值越接近于0，表示当前网络输入的识别结果与字符i越接近。

AlexNet(2012)

Alex Krizhevsky(Geoffrey Hinton)
ILSVRC12冠军

特点:

  1. 使用ReLU激活函数
    收敛速度比sigmoid/tanh快很多

  2. 防止过拟合：Dropout、Data augmentation、重叠池化(s < k)

LRN: Local Response Normalization
模拟生物学上的侧抑制(lateral inhibitio),被激活的神经元会抑制相邻神经元。

有助于增加泛化能力。

  1. 其它: 多GPU训练, LRN层

两块GTX580 GPU

8层(5层卷积+3层全连接)
